ISP

My Team's ISP Project was based on the game 2048.

Conclusion

We expected that the rendering of the tiles was going to be the hardest, but that was easily done with

a function rendering all the tiles from an array keeping the positions of the current tiles. It turns out, the gamesense was

the hardest due to bugs such as allowing the right-most tile being able to combine with the next row's left-most tile due to the array

not accounting for rows in the game.
